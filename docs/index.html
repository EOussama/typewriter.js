Description

Upon making a portfolio of my own, I had this idea that sounded very cool in my head at the time, where I could make an intro that outputs a certain headline in an old typewriter fashion, and that's what this javascript file is for, simply, just that, you specify the output element, play with the parameters you have, and let the magic happen. Nothing too shabby to be honest, just another handy tool for you, and there are a lot nowadays, too excited to start up? check out the installation guide below...

Installation

Steps

To start using this, and I know you are, hopefully, take in account thefollowing steps

  1. Download the latest version here.
  2. Unzip the package somewhere.
  3. Copy the typewriter.js or the typewriter.min.js to your scripts file.
  4. Copy the sounds folder to your project.
  5. Make sure both your typewriter js file's parent folder and sounds folder are part of the same parent directory.

Project structure

- project/ |________ |- scripts/ |________________ |- typewriter.js |- whatever.js |- styles/ |- sounds/ |- images/ |- someFolder/ |- index.html

Documentation

Before resulting in any animation, you first need to have a (you-guessed-it) typewriter, just like in the real world, where can you get one? you simply have to create typewriter object by instantiating the TypeWriter constructor. That's when you get to pass in the animation parameters, and only then, you can make the typewriter make its own thing, that is, animation the text, as a starting point, let's break down our main constructor.

The constructor

TypeWriter ( { target = 'undefined', text = target.textContent.trim(), time = 30, audio = false, cursor = { activated: false, type: 1 } } )

In case you don't want to go through all of the parameters, you can skip them all, as they all have a default backup value to operate, but, the same cannot be said about the target property, you must pass in a valid DOM element, or else, it wouldn't work, the following table explains all of the properties mentioned above in a better manner;

Property Description Default value
target The element you want to perform the animation on, must be a valid DOM element. undefined
text The text you want to be output as a result of the animation The target's text (textContent)
time The time it would take to output a single character 30 milliseconds
audio Whether or not to make use of some sound effects to compliment the typing animation false
cursor An object with 2 properties, the first indicates whether to use a cursor effect at the end of the output text, and the second is for the type of the cursor to display (“_” and “|”) {activated: false, type: 1}

Instantiating an object

Now that we've explained what each of the properties do, here comes the instantiation part, simply provide a variable to hold the returned value upon calling the constructor;

var tw = new TypeWriter ( { target: document.getElementById('sample'), text: 'Hello, there!', time: 200 } ); tw.type({loop: true});

The code that we've just written, of course after a thousand prayers that it would work at the first shot, outputs the following...

Well, all we did, was create a typewriter object, pointed it to the element with the id of sample, assigned the text to output (Hello, there!), making it so that each character is output every 200 milliseconds.

That alone only created a mere object, then we used the type method, and passed in the loop argument setting it to true in order for the animation to loop, and surprise, surprise, that isn't the only method available, let's introduce you to your new beloved friends.

Methods

There are quite the variety of methods you can use to both animate, and change the typewriter's option on run time:

volume
Description

This fellow is a special case, it's not a method per say, this is a getter/setter(using modern js to sound cool) for controlling the typing sound globally (if activated of course), it accepts only one argument, and must be between 0.0 and 1.0, anything lower or bigger will be auto-wrapped. It's necessary that you use this on the TypeWriter class directly, as it's a static property.

Syntax
Setter
volume(vol)
Argument Type Default value Description
vol float none The value of the volume you want to set
Usage
TypeWriter.volume = 0.4;

Basically, that reduces the volume of the typing sound effects and sets it to 0.4.

Getter
Usage
TypeWriter.volume;

This would return the current global volume for the typing effect.

type
Description

This is the primitive method that does it all (mostly), a single call, and all is put to motion, it accepts an object of five arguments, although, in 99% of the time, you'd find yourself not having to pass any, or at least only the first one.

Syntax
type({callback = () => {}, start = 0, chars = this.text.trim().length, delay = 0, loop = false})
Argument Type Default value Description
callback function none Any function you want to execute upon finishing the animation
start integer 0 The exact position (character) you want the animation to start from
chars integer The amount of characters there are on the targeted element's text The number of characters you want to output, and thus where the animation stops at
delay integer 0 The time (in milliseconds) before the animation starts
loop boolean false Whether the animation should loop or not
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.type({ callback: function() { alert('Animation finished'); } });

The code above, instantiate a new typewriter object and calls the type function on it in order to start the animation, that will alert the user with the message of (Animation finished) when the animation is finished.

delete
Description

This function performs a script deletion animation, basically, if the type function is light, this one would be the dark, it simply does the same thing, just in reverse.

This function accepts an object with three arguments, however, you can still call it without any, that will cause the whole text to be deleted.

Syntax
delete({callback = () => {}, chars = this.text.trim().length, delay = 0})
Argument Type Default value Description
callback function none Any function you want to execute upon finishing the animation
chars integer The targeted element's text's length The number of characters you want to delete, and thus where the animation stops at
delay integer 0 The time (in milliseconds) before the animation starts
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.delete({ chars: 10, callback: function() { alert('Animation finished'); } });

The code above performs a deletion animation on the element's text, where only the last 10 characters are deleted, then, the passed callback will execute.

stop
Description

Freezes or strops the animation completely at anytime, to be used in care.

Syntax
stop()
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.type(); setTimeout(() => { tw.stop(); }, 500);

The animation will start then stop after half a second.

pause
Description

This one freezes the animation too, but the difference between it and the previous method stop, is that this one can is just temporary, while the other is absolute, animations that are frozen with the stop method, cannot be resumed, and started from where they left out, and thus must reboot from the beginning, but this method, provides us with the ability to resume the animation, using the method we will get to see next...

Syntax
pause()
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.type(); setTimeout(() => { tw.pause(); }, 500);

Exactly like the previous example, the animation will start then freeze after half a second, and westill have the chance to start it again, how cool is that?

resume
Description

We teased this one on the previous section after an animation is frozen using the pause method, this method can start it out from where it left out, resuming the animation with ease.

Syntax
resume()
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.type(); setTimeout(() => { tw.pause(); }, 500); setTimeout(() => { tw.resume(); }, 1000);

We start an animation, pause it after half a second had passed, then resume it after another half a second.

setText
Description

You can think of this like a setter (as they are called in OOP), where you can set the animation text, and it works just as good in real time, while the animation is running.

Only a single argument is passed in this one, that is the text you want to set, simple enough.

Syntax
setText(text = this.target.textContent)
Argument Type Default value Description
text string The target's text The text you want to be output with the animation
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.setText('Lorem ipsum dolor sit amet.');

We've instanciated a typewriter object, but without specifying the output text, that by default will have the text be the target's text content, then, we used the setText function in order to change that to Lorem ipsum dolor sit amet.

setCursor
Description

Used to change the typewriter object's cursor, calling this with no parameters will toggle the cursor while calling it with the parameters, that makes for two, gives more control over the animation's cursor. the argument has to be an object with two properties for this to work properly.

Syntax
setCursor({activated = !this.cursor.activated, type = 1})
Argument Type Default value Description
activated boolean The opposite of whatever value is currently set Whether or not to use a cursor effect
type integer 1 The type of the cursor to use, there are only two available “_” and “|”, that's 1 and 2 respectively, anything other than those will have the type be set to 1 nevertheless
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.setCursor({ activated: true, type: 2 });

That will toggle the cursor on, and set the type to 2, “|”.

setAudio
Description

Used to toggle the typing sound effect on and off, works in real time too, it accepts a single argument, that is a boolean of true or false to indicate turning the effect on and off respectively, however, passing no arguments will have the audio be toggled to whatever the opposite value that's currently set.

Syntax
setAudio(audio = !this.audio)
Argument Type Default value Description
audio boolean The opposite of whatever value is currently set Whether or not to use a typing sound effect to compliment the animation
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.setAudio(true);

That instanciates a typewriter object and enables the typing sound effect.

setTime
Description

Used to change the output per character time, by default, it's 30 milliseconds, but this allows to slow/speed it down anytime, works in real time as well.

Syntax
setTime(time = 30)
Argument Type Default value Description
time integer 30 The time to wait before outputing another character
Usage
var tw = new TypeWriter({ target: document.getElementById('someId') }); tw.setTime(200);

That changes the animation output time to 200 milliseconds, which is reasonable enough. Resulting in outputting a character every 200 milliseconds, pretty slow if you asked me.
